# 🚀 My Portfolio

A modern, responsive **developer portfolio website** built using React, Vite, and Tailwind CSS.
Designed with a clean UI, smooth animations, and a futuristic black + neon green theme.

---

## 🌐 Live Demo

👉 https://your-site-name.netlify.app

---

## 🛠️ Tech Stack

* React (Vite)
* Tailwind CSS
* Framer Motion
* Lucide React Icons

---

## ✨ Features

* ⚡ Modern and responsive design
* 🎨 Futuristic UI (Black + Neon Green theme)
* 📱 Fully mobile-friendly
* 🧩 Component-based architecture
* 🎬 Smooth animations (Framer Motion)
* 📊 Skills section with progress bars
* 💼 Projects showcase with hover effects
* 📄 Downloadable CV
* 🔗 Social media integration
